# BudgetByte

A fully functional, zero-cost, real-world expense tracker built with Next.js 14+, Tailwind CSS, TypeScript, and localStorage. Features real-time currency conversion via the Frankfurter API and data visualization with Recharts.

## Features
- Add, view, and delete expenses
- Real-time currency conversion (Frankfurter API)
- Data stored in browser localStorage (no backend)
- Responsive dashboard UI (dark/light mode)
- Visualize expenses by category and date (Recharts)
- Modern SaaS-style design

## Tech Stack
- Next.js 14+ (App Router, TypeScript)
- Tailwind CSS
- Lucide-react (icons)
- Recharts (charts)
- localStorage (persistence)

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

### 3. Deploy to Vercel (One Click)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/project?template=https://github.com/vercel/next.js/)
